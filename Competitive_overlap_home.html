QSR Market Analysis

Sonic Drive-In vs. Wendy's

Final Technical Report

Project Description: This project counts the number of locations of two popular quick-service restaurants in Texas - Sonic Drive-In and Wendy's - and evaluates their competitive market overlap.

Extraction: We collected each brands' address data from their respective websites via the python programming language, as well various libraries, including: BeautifulSoup, requests, etc. After scraping the data, we exported the Wendy's data set into a csv file, then read in the csv in order to fullfill the project requirements.

Transformation: First, we converted both data sets into a Pandas dataframe. As for cleaning the Sonic data, we did the following: (1) Renamed columns headings, (2) Dropped duplicate addresses, (3) Removed extra commoms, and (4) Excluded any non-Texas states. Then, we merged the two separate dataframes with a innerjoin. Next, we utilized a .group_by() method by zipcode. Finally, we used a .count() to calculate the market overlap.

Load: We decided to use MySQL, a relational database, for our production database. This decision was made to take advantage of MySQL's table based database, which stores data in columns and row formats. While there are benefits to non-relational databases, such as cheaper costs and ease of scale, our data fit this option best.
